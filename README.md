# PsicWork - Sistema de Gestão de Recursos Humanos

Um sistema web completo e moderno para gestão de Recursos Humanos, desenvolvido com HTML, CSS, JavaScript e Bootstrap 5.

## 📋 Funcionalidades

### 🏠 Dashboard
- **Indicadores visuais**: Total de funcionários, novos contratados e quantidade de departamentos
- **Cards interativos** com ícones e animações
- **Layout responsivo** com cores corporativas

### 👥 Gestão de Funcionários
- **Cadastro completo**: Nome, CPF, cargo, departamento, salário e data de admissão
- **Listagem em tabela** com paginação e filtros
- **Busca em tempo real** por nome, cargo ou departamento
- **Operações CRUD**: Criar, editar e excluir funcionários
- **Validação de formulários** com feedback visual

### 🏢 Cargos e Departamentos
- **Listagem organizada** de cargos disponíveis
- **Departamentos estruturados** da empresa
- **Interface limpa** para visualização

## 🎨 Design e UX

- **Interface moderna** com gradientes e sombras
- **Ícones Bootstrap** para melhor experiência visual
- **Animações suaves** (hover, transições)
- **Responsivo** para desktop e dispositivos móveis
- **Paleta corporativa**: Azul, branco e cinza
- **Menu lateral** para navegação intuitiva

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com Flexbox/Grid
- **JavaScript (ES6+)**: Lógica interativa
- **Bootstrap 5**: Framework CSS responsivo
- **Bootstrap Icons**: Biblioteca de ícones

## 🚀 Como Executar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/Eduardamacedo01/SistemadeGest-odeRH.git
   cd SistemadeGest-odeRH
   ```

2. **Abra o arquivo** `index.html` no navegador ou use um servidor local:
   ```bash
   # Usando Python
   python3 -m http.server 8000

   # Ou usando Node.js
   npx serve .
   ```

3. **Acesse** `http://localhost:8000` no navegador
